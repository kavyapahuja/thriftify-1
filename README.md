# thriftify1
Thriftify is a thrift shopping mobile app built with Flutter and Dart, optimized for Android. Users can browse curated fashion categories such as Vintage, Y2K, Streetwear, and Retro, sell their own style, and create an account through secure login and sign-up functionality. The app uses Firebase Authentication for user management and Cloud Firestore for storing and managing application data, providing a scalable backend for future marketplace features.
